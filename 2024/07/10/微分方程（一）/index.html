数学建模:微分方程专题(1)

常微分方程和偏微分方程是常见的数学模型,但事实上,这类问题的解析解并不好求,因此考虑从数值的角度解决问题,也就引申为差分方程.
其实,解决问题的话重要的是提出解决问题的方案,也就是把问题表述为微分方程是重要的,至于如何求解,反正也不用手算的啦(bushi)

一.常微分方程ODE(ordinary differential equation)

1.简介

我们不从实例引入,不妨回忆一下高数中经典的常微分方程问题.
一个微分方程问题可以分解为这几个特性:阶数\齐次\线性\常系数,那我们也可以相当,对于一阶最通用的是变量代换,分离变量,齐次,多项式等方法.对于二阶,有齐次情况下的通项公式,再加上非齐次的特解即为通解.
对于一些特殊的高阶常系数微分方程也可以用特殊方法解决,比如伯努利方程和可降阶的高阶微分方程.

3.边界问题

上述的微分方程求解的目标是获得通解,通常会给出0点的值,这个问题就转换为边界问题或者初值问题.这些基础的知识专业课都学过啦就不细说了,着重说一下编程问题QAQ.
课上教使用的是MATLAB,但我对python更喜好一些,当然python计算完成后保存数据再到matlab绘图是中很好的方法,毕竟可视化的绘图方便设计.

二.python求解微分方程

1.符号计算解析解

引入sympy库,定义好符号和等式,使用dsolve求解即可,当然方便起见可以把结果输出为latex公式.
这里详细介绍一下这几个函数

  1. sympy.symbols(name, **args):定义符号,后面的参数可以指定符号的属性,比如real=True即为实数,positive = True即为正数.
  2. Function(name):name是一个字符串,而后面接上()即为参数列表(使用了Function类的__call__方法,具体的实现是会产生一个Aooliable示例保存函数和变量列表)
  3. Eq(lhs, rhs, **options):左式和右式,可选参数自己定
  4. dsolve(eq, func, hint, simplify, ics, **kwargs)函数用于求解微分方程
    1. eq: 表示微分方程的等式对象
    2. func: 微分方程中的未知函数
    3. hint:（可选）用于指定求解微分方程的方法。如果不指定，sympy会尝试使用默认的方法
    4. simplify:（可选）布尔值，表示是否简化结果
    5. ics:（可选）用于指定初始条件或边界条件
    6. **kwargs: 其他关键字参数，用于影响求解过程
  5. sympy.diff(expr, var, n=1, **kwargs):expr方程,var变量列表,n默认为一阶(都是关于变量的偏导,一元可以看成特殊的偏导),也可以连续求偏导,按顺序给出var就可以

2.使用scipy

可以使用scipy.integrate.odeint来解决
其完整的参数形式下所示

通用 余下
scipy.integrate.odeint(func, y0, t) args=(), Dfun=None, col_deriv=0, full_output=0, ml=None, mu=None, rtol=None, atol=None, tcrit=None, h0=0.0, hmax=0.0, hmin=0.0, ixpr=0, mxstep=0, mxhnil=0, mxordn=12, mxords=5, printmessg=0
参数 功能
func 定义微分方程的函数，形式为 func(y, t,*args)，其中 y 是当前状态变量，t 是当前时间，*args 是传递给 func 的额外参数。
y0 初始条件，可以是单个数值或数值数组
t 时间点数组，表示求解微分方程的时间范围
args 元组，用于传递给 func 的额外参数
Dfun 用于指定 func 的雅可比矩阵（Jacobian matrix）的函数，默认为 None
col_deriv 已弃用，默认为 0
full_output 如果为 True，则返回一个包含解和诊断信息的元组，默认为 False
ml 和 mu 分别是低阶和高阶导数的边界对角线的数量，用于配置线性多步方法
rtol 和 atol 分别是相对和绝对误差容忍度
tcrit 时间点的数组，指定需要精确求解的时间点
h0 初步时间步长，默认为 0.0
hmax 和 hmin 分别是最大和最小时间步长
ixpr 如果为 True，则打印有关积分过程的诊断信息，默认为 False
mxstep 最大允许的步数，默认为 0（表示没有限制）
mxhnil 允许的连续零步长的最大次数，默认为 0
mxordn 和 mxords 分别是非刚性和刚性问题的最大顺序，默认分别为 12 和 5
printmessg 如果为 True，则在积分完成后打印诊断信息，默认为 False

可以看出odeint实际上是使用数值方法求解的,第一个参数是微分方程的函数,要求返回列表是各阶微分,第二个参数则是各阶的初值,如果没有初值而是其他值,需要便变量代替.(其实这是通过微分方程组求解的,初值实际上是各个函数的初值,重点在于是如何定义函数返回和初值的)
